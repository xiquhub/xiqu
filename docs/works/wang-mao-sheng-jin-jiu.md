# 王茂生进酒

![cover](/covers/wang-mao-sheng-jin-jiu.jpg)
> 行当/题材：历史人情剧 / 生、旦行

## 剧情简介

本剧取材于闽剧传统剧目《红绫袄》，以唐初卖炭翁王茂生的平凡生活为主线，书写小人物大情怀。

隆冬风雪，卖炭翁王茂生和妻子花嫂生活清苦，却心地善良。某年，王茂生在雪夜偶遇落魄贫困、蜷缩街头的薛仁贵，出于同情将其接入家中，给予温饱。王茂生更以朴素的话语鼓励薛仁贵出仕报国、投军建功。此后薛仁贵参军，屡立战功，被封为平辽王。王茂生夫妇岁月蹉跎，不改清贫，然念及当年的扶助之恩，薛仁贵凯旋之际特地备下盛宴邀请昔日恩人。王茂生家境寒微，无力携重礼，仅以清水充酒登门贺喜，却被薛仁贵视为至味——清水变美酒，昭示着真情胜过一切名利富贵。全剧歌颂王茂生夫妇善良质朴的美德与守望相助的精神。

## 主要角色

- 王茂生：主角，卖炭翁，善良质朴的平民英雄，由陈景桦（知名闽剧演员）主演
- 花嫂（王妻）：王茂生妻子，贤惠持家，由翁雪贞主演
- 薛仁贵：受王茂生帮助的落魄军士，后封平辽王，由杨帅主演
- 薛夫人柳金花：国家一级演员林颖主演
- 乳娘：国家一级演员张建斌主演

## 故事来源 / 源流

取材于闽剧传统剧目《红绫袄》中的薛仁贵故事，薛仁贵（614—683年）为唐代名将，其事迹广泛流传于民间传说、话本及各地戏曲。本剧将关注焦点从英雄人物转向"王茂生"这一社会底层小人物，以人情温暖为核心，由福州市闽剧院创作于2004年。

- 演出单位：福州市闽剧院

- 首演：2005年2月
