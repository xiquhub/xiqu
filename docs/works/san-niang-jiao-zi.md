# 三娘教子

> 行当/题材：家庭伦理戏 / 旦角戏

## 剧情简介

《三娘教子》是中国戏曲的传统经典剧目，全本题名《双官诰》。故事设于明代，讲述儒生薛广经商在外，因友人吞没托付银两，假造棺木谎报其死，致使家中妻妾以为薛广已亡故。大娘张氏、二娘刘氏相继改嫁，唯有三娘王春娥与忠心老仆薛保留守家中，含辛茹苦抚养刘氏所生之子薛倚哥，靠纺织维生，供其读书。

薛倚哥一日在学堂被同窗讥笑为"无母之儿"，赌气归家、不受管教，顶撞三娘。王春娥痛心之下，当场以刀斩断织机上的布匹，以示"机头既断，学业亦断"之决心，斥责倚哥不成材便如断布无用，并以此警示其珍惜来之不易的读书机会。薛保从旁苦劝，母子情感复归和好。此后倚哥发奋苦读，终中状元。与此同时，薛广以军功还家，父子相认，一家团圆。当初改嫁的大娘、二娘欲重回薛家，王春娥以"自种苦瓜自己尝"之言婉拒，彰显守节自立之德。

剧情核心通用，为各剧种共享传统剧目。

## 主要角色

- **王春娥**（旦角）：三娘，贤惠守节，含辛茹苦教养养子的典范形象
- **薛保**（老生/外）：忠心老仆，从旁调解母子纷争
- **薛倚哥**（娃娃生/小生）：薛广之子，受三娘教导后发奋读书，终成状元
- **薛广**（生角）：儒生，旅途遭难后立军功还家

## 故事来源 / 源流

源自明代传奇《断机记》及杨善之《双官诰》，全本《双官诰》取材于《后汉书·冯衍传》，"三娘教子"为其中最精华之一折。该剧自形成以来，汉剧、晋剧、豫剧、秦腔、楚剧、河北梆子等众多剧种均有移植演出，是中国戏曲中流传最广的伦理题材剧目之一。
