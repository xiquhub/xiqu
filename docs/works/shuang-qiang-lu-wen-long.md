# 双枪陆文龙

![cover](/covers/shuang-qiang-lu-wen-long.jpg)
> 行当/题材：武戏 / 历史戏 / 忠义题材

## 剧情简介

《双枪陆文龙》又名《陆文龙》，故事取材于《说岳全传》，背景设于南宋朱仙镇之战期间。陆文龙自幼被金国大将金兀术收为义子，武艺超群，善使双枪，于战场上连败宋军中四员使用双锤的大将，令岳家军深感棘手。

宋营参军王佐为了感化陆文龙、令其归宋，决然自断左臂，佯称是被岳飞所迫害，以此取信于金兀术，从而得以接近陆文龙。王佐借说书讲古的机会，向陆文龙透露其真实身世——他本是宋朝潞安州节度使陆登之子，父亲早已战死，他却长年在不知情的状态下为父仇之敌效力。

陆文龙得知身世真相，内心大受震动，悲痛欲绝，决意伺机脱离金营归宋。然而就在此时，金兀术察觉异动率兵追杀。金兀术义女琼芳与陆文龙青梅竹马，深厚情谊难以割舍，在生死关头，琼芳以身殉情，含笑赴死。陆文龙悲恸欲绝，奋起大杀金兵，抱起琼芳遗体，纵马驰归宋营，完成了从懵懂义子到自觉忠臣的精神转变。

（剧情核心通用，源自越剧版本，闽剧版系移植改编。）

## 主要角色

- **陆文龙**（武生）：金兀术义子，宋朝节度使遗孤，武艺绝伦，善使双枪
- **王佐**（文生）：宋营参军，自断左臂诈降金营，感化陆文龙归宋
- **琼芳**（旦角）：金兀术养女，与陆文龙情深，最终殉情
- **金兀术**（净角/反角）：金国大将，陆文龙养父
- **岳飞**：宋军元帅，出场较少但为故事背景的核心人物

## 故事来源 / 源流

改编自清代小说《说岳全传》中陆文龙相关章节，由越剧1986年首演（越剧版由赵志刚主演），后经1992年改编更名并由绍兴小百花越剧团演出。闽剧版从越剧或其他剧种版本移植改编。
