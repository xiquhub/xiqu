# 告老尚书

![cover](/covers/gao-lao-shang-shu.jpg)
> 行当/题材：公案

## 剧情简介

此剧围绕一桩错判冤案展开。唐汝琳赴京赴考，好友招祥佑设宴为其饯行。祥佑之妹韵玉趁祥佑大醉之机，冒充其妻王氏，借琴音向汝琳挑逗。汝琳由此误以为王氏轻浮不守妇道。韵玉与尚书之子曹文私通，坊间却误传王氏不贞。韵玉与曹文为掩盖真相，图谋毒杀王氏灭口，却阴差阳错毒死了招祥佑。韵玉趁机反诬王氏与人偷情并谋杀亲夫。此时唐汝琳高中状元，被擢升为巡按，奉命审理此案。因昔日曾被琴挑所迷、心中已先入为主地认定王氏不贞，汝琳错判王氏死罪，一场冤案由此铸成。全剧充满误会与反转，以公案为骨架，深刻揭示了先入之见如何导致司法错判，具有强烈的警世意味。

（剧情核心通用，闽剧版本演法见演出历史。）

## 主要角色

- **唐汝琳**：书生，后高中状元，任巡按，因先入之见错判冤案
- **招祥佑**：汝琳之友，被韵玉和曹文误毒身亡
- **王氏**：祥佑之妻，贞洁守节，蒙冤被判死刑
- **韵玉**：祥佑之妹，与曹文私通，设计陷害王氏
- **曹文**：尚书之子，与韵玉勾结，是幕后黑手
