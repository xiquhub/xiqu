# 贬官记

![cover](http://fj.people.com.cn/NMediaFile/2023/0921/LOCAL16952600836904ZZASX2Q7Y.jpg)


> 行当/题材：公案·廉政

## 剧情简介

五品知府郑则清因娶青楼女子张岫玉为妻，被巡案崔云龙弹劾参奏，由此贬官至偏远小县担任七品县令。初到任时，郑则清案牍未稳，便遇一名自称书吏投奔的外乡人，殊不知此人正是微服私访的巡案崔云龙。

郑则清不知对方身份，与之共事，侦办一桩悬疑人命案。两人各怀心思，暗中相互考量、测试。案件侦破过程中，郑则清处处显露清正廉洁、勇于担当的本色，令微服巡案深为折服。崔云龙最终揭明身份，两位正派官员相互理解，化解前嫌，共同彰显了"清官"的人格力量。

全剧以轻喜剧的笔调呈现廉政主题，通过一贬一查的戏剧结构，探讨官场中坚守正道的艰难与价值，风格幽默而不失厚重。

## 主要角色

- **郑则清**：原五品知府，因娶青楼女子为妻遭贬，为人清正廉洁，是全剧主角
- **巡案（崔云龙）**：微服私访的上级官员，表面扮作书吏，暗中考核郑则清
- **张岫玉**：知府妻子，出身青楼，因丈夫遭贬而身陷困境，形象正面

## 故事来源 / 源流

原为梅林戏传统剧目，源出民间故事《郑青天判案》。1996年由吴永艺据梅林戏同名剧目移植改编为闽剧版本，由福建省实验闽剧院演出。此后评剧亦有同名改编版本。

## 闽剧演出记录
