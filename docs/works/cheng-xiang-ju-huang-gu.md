# 丞相拒皇姑

> 行当/题材：历史

## 剧情简介

《丞相拒皇姑》讲述了北宋年间，宋仁宗欲将皇姑许配给丞相庞洪。庞洪为人刚正不阿，认为自己与皇姑年龄差距悬殊，且皇姑刁蛮任性，配之于自己有损朝廷体面，更恐其搅乱家宅，遂坚决推辞。皇姑闻讯大怒，前来丞相府兴师问罪，并企图以权势压人。庞洪不畏强权，据理力争，陈述利害，最终令宋仁宗深思，打消了赐婚的念头。此剧通过对庞洪忠直、皇姑跋扈的刻画，展现了朝堂上的政治斗争与个人气节的较量。

## 主要角色

- **庞洪**：北宋丞相，为人刚正不阿，忠直敢言。
- **皇姑**：宋仁宗的皇姑，性格刁蛮任性，仗势欺人。
- **宋仁宗**：北宋皇帝，但剧中戏份相对较少，主要推动剧情。
