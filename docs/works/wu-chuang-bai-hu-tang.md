# 误闯白虎堂

![cover](https://puui.qpic.cn/vpic_cover/m3367zd4wlw/m3367zd4wlw_1671766709_hz.jpg)


> 行当/题材：英雄武侠剧 / 武生行

## 剧情简介

本剧改编自《水浒传》中林冲误闯白虎堂的经典故事，剧情核心通用。

北宋年间，东京八十万禁军教头林冲，为人忠厚正直、武艺超群。高太尉之子高衙内垂涎林冲之妻，屡加纠缠未果。高俅遂与陆谦合谋，诱骗林冲进入高太尉府中的白虎节堂——此乃兵器重地，寻常将士不得擅入。林冲不知是计，手执宝刀入内，被早已埋伏的高俅之人当场拿获，以"擅闯白虎堂"之罪诬告下狱。此后林冲蒙冤，被刺配沧州，一路上险遭杀害，最终走上梁山，成为水泊英雄之一。

本剧聚焦"误闯白虎堂"这一关键折子，着力刻画林冲从忠良到被迫落草的心路转变，以及封建权贵迫害忠良的黑暗现实。

## 主要角色

- 林冲：主角，禁军教头，忠厚善良，被迫走上反抗之路
- 高俅：奸臣，陷害林冲的主谋
- 高衙内：高俅之子，骄横淫邪，林冲受难的直接起因
- 陆谦：林冲好友，被收买后出卖林冲

## 故事来源 / 源流

源自施耐庵（或元末明初）《水浒传》第七至九回"豹子头误入白虎堂"等章节，是《水浒传》中最广为人知的情节之一。京剧、豫剧、越剧、闽剧等众多剧种均有改编，《误闯白虎堂》《林冲夜奔》《野猪林》等为常见的折子戏或全本演出名目。
