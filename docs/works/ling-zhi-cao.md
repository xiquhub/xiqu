# 灵芝草

> 行当/题材：神怪

## 剧情简介

（剧情核心通用）本剧应属白蛇传系列传统剧目，取材于"盗仙草"折子。白娘子为救中毒的许仙，只身赴昆仑山（或峨眉山）盗取能起死回生的灵芝仙草，途中与鹿童、鹤童激战，历经险阻，最终取回仙草，救活夫君。

## 主要角色

- **白娘子（白素贞）**：女主角，白蛇精，为救夫盗取灵芝
- **许仙**：白娘子夫君，因故中毒需灵芝救治
- **鹿童 / 鹤童**：守护仙草的神兽童子
