# 春草闯堂

> 行当/题材：才子佳人（喜剧）

## 剧情简介

相国千金李半月与贴身丫鬟春草进香途中遭遇吏部尚书之子吴独的无赖纠缠，幸被路过的义士薛玫庭解围。此后吴独打死渔家女张玉莲，薛玫庭义愤填膺，将其击毙，随即主动投案自首。为救薛玫庭，机智勇敢的丫鬟春草闯进公堂，声称薛玫庭乃相府姑爷，迫使知府胡进不敢草率处置。胡进赶赴相府对证，情急之下，小姐半月假意认薛为未婚夫。岂料相国李仲钦在京闻讯，急书命胡进取薛玫庭首级解京。春草与半月趁机巧改书信，反令胡进大张旗鼓地护送薛玫庭进京，抵京之时百官齐贺，皇帝御赐匾额，李相国骑虎难下，只得认薛为婿。全剧以丫鬟春草为核心，通过"闯公堂""涂改书信"等一系列喜剧性情节，展现下层人物的机智与正义感，是中国戏曲十大喜剧之一。

（剧情核心通用，闽剧版本演法见演出历史。本剧原为莆仙戏，后被全国600余剧团移植演出，闽剧亦有移植版本。）

## 主要角色

- **春草**：相府丫鬟，主人公，机智勇敢、嫉恶如仇，全剧的灵魂人物
- **李半月**：相国独生女，温婉识大体，与春草共同谋划救人
- **薛玫庭**：游侠义士，见义勇为，击毙恶少后主动投案
- **胡进**：知府，被春草和半月牵着鼻子走的喜剧反衬人物
- **李仲钦**：当朝相国，李半月之父，最终不得不接受既成事实
- **吴独**：吏部尚书之子，恶少，打死民女被薛玫庭击毙

## 故事来源 / 源流

改编自福建莆仙戏传统剧目《邹雷廷》，由陈仁鉴（执笔）、柯如宽、江幼宋整理改编，将原剧中的男仆李用改为女婢春草，增强了喜剧效果。1960年首演即获好评，1979年进京参加国庆三十周年献礼演出引起轰动，获剧本创作一等奖、演出一等奖，被列为中国当代十大喜剧之一。

## 备注

《春草闯堂》是典型的跨剧种剧目，在莆仙戏首演后被京剧、高甲戏、花鼓戏等600余个剧团移植。闽剧与莆仙戏同属福建地方戏曲，该剧在闽地流传甚广。国家京剧院版由刘长瑜饰春草，已成经典版本。
