# 天波府

> 行当/题材：历史英雄剧 / 生、旦、净行

## 剧情简介

天波府是北宋抗辽名将杨业（杨令公）的府邸，位于东京开封城西北天波门旁，宋太宗御书"天波府"匾额以褒奖杨家忠烈。本剧以天波府为核心场景，演绎杨家将的忠义故事，剧情核心通用。

杨令公率子出征抗辽，屡建战功，然遭奸臣潘仁美陷害，于狼牙谷被困，撞李陵碑壮烈殉国。八子中，大郎、二郎、三郎、七郎皆战死，四郎被俘入辽，五郎出家，唯六郎杨延昭继承门风守边疆。天波府中，佘太君（折氏）统领杨门女将，穆桂英挂帅出征，薪火相传，演绎一代代忠臣报国的悲壮史诗。本剧或截取其中某一折——如"辕门斩子""佘太君出征"等经典场次——集中呈现天波府的家国情怀。

## 主要角色

- 佘太君（折氏）：杨业之妻，杨门女将领袖
- 杨令公（杨业）：抗辽名将，壮烈殉国
- 杨六郎（杨延昭）：六子，继承父志守边关
- 穆桂英：杨宗保之妻，杨门女将代表
- 潘仁美：奸臣，陷害杨家的反面人物

## 故事来源 / 源流

源自北宋将领杨业家族的历史事迹，经宋元话本、明代熊大木《北宋志传》及清代《杨家府演义》反复加工，形成系统的杨家将传说体系。相关戏曲剧目众多，如《四郎探母》《李陵碑》《杨门女将》《穆桂英挂帅》等，在京剧、豫剧、闽剧等各剧种均有广泛演出。
