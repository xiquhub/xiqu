# 孟宗哭竹

![cover](https://i.ytimg.com/vi/flxGANTHkzc/hqdefault.jpg)


> 行当/题材：旦/孝义道德教化

## 剧情简介

三国时期吴国，江夏人孟宗自幼丧父，与年迈病弱的母亲相依为命。母亲某冬身染重疾，大夫嘱咐须以鲜竹笋煮汤调养。时值隆冬，天寒地冻，竹笋全无。孟宗心急如焚，独自跑入竹林，双手抱竹，泪如雨下，向天地倾诉无力奉亲之苦。其一片孝心感动天地，地面忽然裂开，竹根处破土涌出数茎嫩笋。孟宗大喜，采归烹汤奉母，母亲饮后果然痊愈。后来孟宗官至大司空，被后世传颂为二十四孝之一。

闽剧版本以福州方言演唱，突出孟宗寒冬求笋、父母面前难尽孝道的情感张力，将原本简短的孝道故事扩展为生动的折子戏，以小调与逗腔衬托悲凄哭诉的情绪，最终以奇迹降临收结，宣扬孝道感天动地的主旨。整出戏情节单纯而情感浓烈，是传统伦理教化戏的典型代表。

## 主要角色

- **孟宗**：孝子，剧中核心人物，以生角或小生应工
- **孟母**：年老病弱，旦角，是孟宗行孝的对象

## 故事来源 / 源流

本剧改编自中国传统"二十四孝"故事之第十七则"哭竹生笋"。故事最早见于魏晋文献，孟宗为三国吴国真实历史人物，后世文人将其孝行收入《二十四孝图》并广为传播。同题材在京剧、川剧、潮剧等多个剧种均有折子戏版本。
