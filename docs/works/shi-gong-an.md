# 施公案

![cover](http://puui.qpic.cn/vpic_cover/h0033302ffi/h0033302ffi_hz.jpg)


> 行当/题材：侠义公案戏

## 剧情简介

《施公案》取材于清代同名小说，故事背景为清朝康熙年间。循吏施世纶（字文开，号浔江）为人廉洁刚正，在黄天霸等一众江湖豪杰的辅助下，屡次惩治地方贪官污吏和土豪恶霸，为百姓申冤昭雪。

剧目通常截取小说中具体案件改编，以施公问案为主线，黄天霸以武力协助缉拿凶犯，展现"文官断案、武侠助力"的侠义公案模式。施世纶以"外科圣手"之誉断案如神，刚正不阿，面对强权依然据理力争；黄天霸则是"武艺超群、义薄云天"的江湖豪杰，二人一文一武相辅相成，共同维护地方安宁。

《施公案》是中国古代侠义公案文学中将"公案"与"侠义"两大主题合流的先驱之作，深刻影响后世的《三侠五义》《彭公案》等作品，并衍生出众多戏曲剧目。

（剧情核心通用，源自清代小说，改编为闽剧版本。）

## 主要角色

- **施世纶**（施公）：廉洁刚正的地方官员，善于断案
- **黄天霸**：江湖豪杰，武艺高强，辅佐施公缉拿恶人
- 贪官、土豪、被害百姓等配角

## 故事来源 / 源流

改编自清代同名白话小说《施公案》，该书最早以说唱形式在民间流传，嘉庆三年（1798年）形成97回本，后经历代续书，至光绪年间累积成528回本。历史原型为清朝康熙年间扬州知府施世纶，史料记载其为人清廉、深得民心。京剧中亦有多个与《施公案》故事相关的剧目。
