# 乐昌破镜


> 行当/题材：才子佳人

## 剧情简介

（剧情核心通用，源自著名历史典故"破镜重圆"，多个戏曲剧种有同题改编，闽剧版细节未见专项记录。）

南朝后主陈叔宝沉湎享乐，北方隋朝虎视眈眈。陈叔宝之妹乐昌公主貌美才高，与驸马徐德言（太子舍人）情深意笃。徐德言深知陈国危在旦夕，二人商定：将铜镜摔成两半，各执一半，若国破后失散，每年正月十五于街市叫卖半面铜镜，以此为识。

陈国不久为隋所灭。乐昌公主被俘入长安，成为隋朝重臣越国公杨素的侍妾。徐德言一路颠沛流离，辗转来到京城，正月十五持半镜上街叫卖，偶遇同样在街上叫卖半镜的乐昌公主家仆，两半铜镜严丝合缝。徐德言在镜上题诗："镜与人俱去，镜归人未归。无复嫦娥影，空留明月辉。"

杨素得知此事，被夫妻情深所感动，乐昌公主亦如实相告，诉说二人约定的始末。杨素心生感慨，设宴款待徐德言，翌日亲自送乐昌公主归还夫君，并厚赠钱财，成全这对历经离乱的夫妻。"破镜重圆"一词由此传世。

## 主要角色

- **乐昌公主**：南朝陈叔宝之妹，才貌双全，以半镜为约，坚守夫妻之情
- **徐德言**：乐昌公主之夫，太子舍人，才学过人，持镜千里寻妻
- **陈叔宝**：南朝后主，荒淫亡国
- **杨素**：隋朝重臣，越国公，因感动于乐昌公主夫妻情深而成全二人

## 故事来源 / 源流

源自南朝至隋代之间的历史典故，最早见于《本事诗》等唐代笔记，后广泛流传于诗词、小说与戏曲。元代沈和甫有杂剧《徐驸马乐昌分镜记》，明代传奇《合镜记》《金镜记》亦以此为蓝本，"破镜重圆"成为汉语中描述离散夫妻复合的经典成语。
