# 哑女鸣冤

![cover](http://www.fzmjtc.cn/__local/A/FC/D0/A00B8F933AB868DF3B5AA39C807_40F6DDCC_1E1E2.jpg)


> 行当/题材：苦情公案戏 / 青衣、花旦

## 剧情简介

故事设定于明朝。书生陈光祖家道中落，投奔岳父掌家，却遭继室排斥，被扫地出门。未婚妻掌上珠深明大义，暗中接济陈光祖，助其刻苦攻读。陈光祖终于高中，衣锦荣归之日，继室却设下阴谋，以次女掌赛珠冒名顶替上珠，强行与陈光祖成婚。

善良的掌上珠遭继室纵火陷害，侥幸得呆大（家仆）相救，然而继室又买通歹人，将掌上珠毒哑并断去手指，令其无法言诉冤情。上珠百折不挠，在呆大护送下历尽艰辛赶赴京城告状。途中屡遭阻挠，在忠仆掌忠等人的帮助下，冤情终于大白于天下。奸人掌赛珠受到应有惩处，陈光祖与掌上珠终得团圆，哑症亦奇迹复愈。

全剧以哑旦表演见长，无言诉冤的舞台呈现极具感染力，是各剧种争相移植的经典苦情戏之一。

## 主要角色

- **掌上珠**（青衣/哑旦）：书生未婚妻，受尽迫害仍坚持伸冤，全剧核心。
- **陈光祖**（小生）：书生，高中后被奸人利用，最终主持公道。
- **掌赛珠**（花旦/反派）：继室之女，冒名顶替，行恶多端。
- **呆大**（丑行）：忠实家仆，护送哑女进京，喜剧担当。
- **掌忠**（老生）：忠仆，助力申冤。

（上述角色名参考淮剧原版，闽剧版本或有调整。）

## 故事来源 / 源流

原为淮剧连台本戏，曾一度濒临失传。1980年上海淮剧团于卢湾区文化馆整理首演，由筱海红主演掌上珠，轰动上海。此后被扬剧、黄梅戏、锡剧、越剧、粤剧、闽剧、京剧、秦腔等数十个剧种、近百个剧团移植上演，成为1980年代以来流传最广的苦情公案戏之一。

## 闽剧演出记录

闽剧版《哑女鸣冤》系移植自淮剧《哑女告状》，为福州及闽东各地剧团常演剧目之一。具体主演信息暂无详细资料（来自 batch-08 文件，leads/troupe 字段为空）。
