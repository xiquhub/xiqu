# 孟丽君

![cover](https://img1.fjdaily.com/app/images/2023-01/20/7d8706a8-0dd0-4968-af37-896b2c96e518.jpg)


> 行当/题材：才子佳人

## 剧情简介

才女孟丽君因父亲蒙冤，家遭株连，被迫女扮男装出走，行前自画肖像赠与未婚夫皇甫少华，以明心迹。孟丽君改名郦君玉，凭借才学连中科举，官拜丞相，巾帼胜须眉。

五年后，皇甫少华御敌立功班师回朝，皇帝欲赐婚他人，少华誓言非孟丽君不娶，并经多方查探，发现郦丞相正是孟丽君。然孟丽君深知女扮男装欺君之罪足以累及夫家，故不肯相认。皇甫少华情急之下，献上肖像泄露秘密，引发了皇帝觊觎郦丞相的风波。

孟丽君以机智与皇帝周旋，施金蝉脱壳之计，更请出太后出面调停，终使太后以皇室江山为重，亲自斡旋，昭雪孟家沉冤，惩办奸佞。孟丽君与皇甫少华终成眷属，大团圆收场。

## 主要角色

- **孟丽君（郦君玉）**：女主角，才貌双全，女扮男装高中丞相；谢婉萍、孙建斌先后饰演
- **皇甫少华**：孟丽君未婚夫，武将；杨帅饰（获第八届全国戏剧文化奖表演金奖）
- **皇帝**：垂涎孟丽君，被太后制止
- **荣兰**：孟丽君丫环；蔡加圻饰
- **权昌**：奸臣；林亿惠饰

## 故事来源 / 源流

改编自清代才女陈端生所著弹词《再生缘》，创作于乾隆三十三年至四十九年（1768—1784年），全书十七卷近六十万字，后由梁德绳续写。故事背景设于元代，是中国古典文学中女性意识觉醒的代表性作品，该题材被越剧、闽剧等多个剧种改编演出。
