# 红裙记

![cover](https://www.cflac.org.cn/zgwl/dfwl/sheng/fujjian/201310/W020131024608239482443.jpg)


> 行当/题材：家庭

## 剧情简介

此剧取材于清代发生在福州台江上下杭的真人真事。书生王成龙嗜赌成性，不仅让妻儿衣食无着，更将岳家所赠的一条红裙输作赌资。败尽家财后，他绝望地投水闽江，幸被陕商许希和救起，带往长安。妻子柳氏误以为丈夫已亡，独自含辛茹苦守节抚孤，一守便是十三年，其子王达官长大后每日上街卖饼维生。王成龙在长安入赘许家，后奉许家命令返回福州经商，偶然在吴二伯船行遇见卖饼的王达官，方知其为亲生骨肉。他辗转来到故宅，亲情与愧疚交织，最终留下房契、白银和那条旧红裙作为补偿，黯然离去。全剧以一件红裙为线索，串联起家庭悲欢与人性抉择，充满浓郁的福州市井气息。

## 主要角色

- **王成龙**：嗜赌书生，酿成家庭悲剧，代表男性在责任与自我放逐之间的两难
- **柳氏**：王成龙之妻，坚韧守节十三年，是全剧道德支柱
- **王达官**：王成龙之子，靠卖饼为生，是家庭苦难的见证者

## 故事来源 / 源流

取材于清代福州台江上下杭的真实民间故事，"王达官卖饼"的传说在当地市井间广泛流传。现行演出本由知名闽剧编剧王仁杰改编，福州市艺术学校历时五年打磨，2009年首演，之后成为多家职业剧团的常演保留剧目，演出逾百场。
