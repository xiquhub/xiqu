# 冯梦龙断案

> 行当/题材：公案

## 剧情简介

该剧取材自明代著名文学家冯梦龙（1574—1646）晚年出仕寿宁知县时的断案事迹。冯梦龙以文名著天下，却在晚年以实际行动践行其文学中所倡导的公正精神。他到任寿宁后，即发现当地土豪劣绅横行，欺压百姓。冯梦龙利用出其不意的时机，亲率人马将为害一方的恶绅绳之以法，并公正处理百姓各类纠纷。全剧以冯梦龙这一历史文人形象为中心，通过多个案件展现其智慧与清廉，彰显明代地方官员主持正义的主题。闽剧创作该题材具有独特的地域意义，因冯梦龙曾在福建寿宁任职（寿宁县今属宁德市），是具有福建背景的历史人物。

## 主要角色

- **冯梦龙**：主角，明代文学家，寿宁知县，以智慧和正直著称
- **陈伯进**（或类似人物）：地方土豪劣绅，欺压百姓的反派人物
- **百姓**：受冯梦龙庇护的平民群众

## 故事来源 / 源流

取材自明代文学家冯梦龙出仕福建寿宁知县（约1634—1638年）期间的历史事迹，见于其所著地方志《寿宁待志》。冯梦龙以《三言》（《喻世明言》《警世通言》《醒世恒言》）闻名，是中国文学史上重要的通俗文学作家。以其为主角的地方戏曲，在闽地具有特殊的文化认同价值。
