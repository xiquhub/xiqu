# 曲判记

## 剧情简介

《曲判记》是闽剧新编古代戏，1981年由林芸生、张哲基、张品生三人联合编剧，同年由福建省闽剧实验剧团首排演出，并参加福建省创作剧目调演。剧名中"曲判"意含冤曲之案遭歪判，或隐指断案之迂曲周折。

全剧围绕一桩冤案展开，主人公黄愿亭饰演的核心角色在错综复杂的案情中周旋，历经波折，最终伸张正义。剧中融入闽剧典型的唱腔表演程式，兼具跌宕剧情与唱腔艺术，展示了1980年代闽剧创作的较高水平。该剧在省级调演中斩获剧本创作二等奖、演出二等奖、导演奖、舞台美术奖、音乐奖共五项奖项，是彼时闽剧创作的重要收获之一。

主演黄愿亭、林聪中、李少华均为省闽剧实验剧团骨干演员，三人搭档演绎了这部兼顾艺术性与思想性的公案新剧。
