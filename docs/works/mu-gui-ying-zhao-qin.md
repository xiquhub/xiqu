# 穆桂英招亲

![cover](http://u2.iqiyipic.com/image/20231029/94/a2/pv_3150606639504200_d_601_1248_720.jpg)


> 行当/题材：刀马旦、武生／英雄儿女

## 剧情简介

北宋时期，辽邦萧太后摆设天门阵，大宋军队久攻不破，杨元帅手下谋士献策，须取穆柯寨的"降龙木"方可破阵。老将杨宗保奉命攻打穆柯寨，与女寨主穆桂英交手，杨宗保武艺精熟却终究力敌不过，被穆桂英生擒上山。

穆桂英见杨宗保英俊勇武，顿生爱慕，欲与之缔结婚姻。杨宗保为臣为将，不敢擅自做主招亲，穆桂英豪气不减，提出以献降龙木破天门阵为条件，恳请杨宗保答婚。杨宗保权衡大局，应允成亲。二人成婚后穆桂英主动请缨，随夫下山助宋抗辽，大破天门阵，为宋朝立下赫赫战功，由此开启杨门女将抗辽的一系列故事。

闽剧版本以福州方言演绎这段英雄儿女情，重点在武打与儿女情长的交织呈现，刀马旦与武生的对打身段是全剧最具观赏性的部分。

## 主要角色

- **穆桂英**：穆柯寨女寨主，刀马旦应工，武艺超群，胆识过人
- **杨宗保**：杨延昭之子，武生应工，北宋将领
- **穆羽**：穆桂英之父，穆柯寨寨主（部分版本出场）

## 故事来源 / 源流

故事脱胎于明清长篇小说《杨家将演义》，穆桂英、杨宗保均为小说虚构人物，以北宋抗辽名将杨业家族为历史原型。此题材在京剧、越剧、豫剧、潮剧等多个剧种均有版本，闽剧《穆桂英招亲》为其闽地移植改编版。
