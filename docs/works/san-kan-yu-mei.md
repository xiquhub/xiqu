# 三看御妹

> 行当/题材：古装喜剧 / 爱情戏

## 剧情简介

《三看御妹》是一部传统古装喜剧，故事背景设于明朝嘉靖年间。将军刘天化之女刘金定，因平定叛乱、救父有功，被皇帝封为御妹娘娘，并下旨告示百姓不得窥看御妹，违者斩首。尚书封雷尚之子封加进慕名御妹英雄风采，极想一睹芳容，遂趁其赴东狱庙拈香之机，乔装乡民"张小二"躲于神台之下偷看，不料被御妹发现。

御妹回府后，父亲刘天化得知此事，严加训斥。御妹心中闷闷不乐，细思种种情景，断定那"张小二"绝非普通百姓，而是冒死前来一睹之人，深感其胆识过人，不由生出爱慕之情，渐染相思之疾。

刘天化奏明皇上，张榜求医。封加进二见御妹之后愈发思念，再次揭榜假扮郎中，瞒过刘天化耳目，进入深宫为御妹诊病；次日复诊，二人第三次相遇，情意渐深。最终皇帝得悉原委，赦免封加进，并赐其与御妹成婚，一段奇缘喜成眷属。

（剧情核心通用，源自越剧传统剧目，已有百年以上演出历史。）

## 主要角色

- **刘金定**（旦角）：御妹，英姿飒爽的女将军之女，被封为御妹娘娘
- **封加进**（生角）：尚书公子，胆识过人，三度冒险得见御妹
- **刘天化**（老生）：将军，刘金定之父

## 故事来源 / 源流

该剧源自越剧传统剧目，在越剧"小歌班"时期由绍兴大班移植而来，已有百年以上演出历史。此后评剧、锡剧、粤剧等多个剧种均有改编版本。1962年香港长城电影公司曾摄制同名彩色电影；1986年上海越剧院摄制电视剧版。闽剧版系从越剧移植改编。
