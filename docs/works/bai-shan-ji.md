# 白扇记

![cover](https://puui.qpic.cn/vpic_cover/z0824z4kas6/z0824z4kas6_hz.jpg)


> 行当/题材：家庭·冤案·孝义

## 剧情简介

知府胡开思告老还乡，途经洞庭湖时被强贼赵大所害，财物被劫，其妻黄氏与女儿金莲被迫为强贼妻妾。危急之中，黄氏将刚出生的儿子用红毡裹绑，抛入湖中托付天命，孩子幸得渔翁刘金秀救起，取名"渔网"，养育成人。

渔网成年后离家寻访生母，途中与典当商人结为义兄弟，不料此人正是当年杀父害家的仇人赵大。端阳节，渔网无心观龙船，独自哀叹身世，被金莲所听，金莲将情况转告黄氏。黄氏借饮酒之机盘问，终于认出渔网正是十三年前自己亲手抛入水中的骨肉，母子二人悲喜交加，相认团聚。

黄氏将先夫留存的御赐白纸扇郑重交付渔网，嘱其携扇进京向外公求援，调兵前来报仇雪恨，以还一门清白。白扇既是身份的凭证，也是一门血仇与希望的象征。

闽剧中此剧系"洋歌"声腔代表性剧目之一。

剧情核心通用，闽剧版本演法见演出历史。

## 主要角色

- **渔网**（胡金元）：被抛弃的孤儿，经历磨难后认祖归宗，担起报仇雪恨的重任
- **黄氏**：知府之妻，忍辱负重，以白扇为信物等待骨肉重逢
- **金莲**：黄氏之女，在虎穴中默默维护家人，传递关键信息
- **赵大**：反面人物，杀人越货、横行霸道的强贼
- **胡开思**（死于故事开场）：知府，是仇恨故事的起点

## 故事来源 / 源流

传统剧目，属公案与家庭题材结合的经典类型，在多个地方剧种中均有流传，包括闽剧、黄梅戏等版本。闽剧版本以"洋歌"唱腔演绎，是该声腔的代表性剧目之一。
