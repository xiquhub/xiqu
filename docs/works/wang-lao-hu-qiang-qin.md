# 王老虎抢亲

![cover](https://so1.360tres.com/t013b177e10f2abcb45.png)


> 行当/题材：爱情喜剧 / 生、旦行（有男扮女装的反串特色）

## 剧情简介

本剧是典型的误会喜剧，以男扮女装为核心噱头，展开一段啼笑皆非的爱情故事，剧情核心通用。

江南才子周文宾对兵部尚书之女王秀英心生爱慕，托人说媒却遭拒绝。好友祝枝山出谋划策，让周文宾在元宵节灯会上男扮女装散心。不料，王秀英之兄王天豹在灯会中倚仗权势拦路抢亲，正好撞上乔装的周文宾，误将其当作美女强行带回府中。入洞房之夜，周文宾与王秀英相遇，两人一夜倾谈，彼此倾慕，私订终身。周文宾回家后苦思良策，王天豹也察觉被戏弄，两方周旋之间，误会层层叠叠，最终真相大白，周文宾堂堂正正娶得王秀英过门，皆大欢喜。

## 主要角色

- 周文宾：才子主角，因男扮女装卷入一场喜剧误会
- 王秀英：王天豹之妹，温柔聪慧，与周文宾情投意合
- 王天豹（王老虎）：兵部尚书之子，外号"王老虎"，莽撞却也可笑
- 祝枝山：周文宾的朋友，出谋献策的喜剧配角

## 故事来源 / 源流

越剧传统剧目，1958年2月由上海合作越剧团首演于瑞金剧场，编剧红枫，导演李卓云，毕春芳饰周文宾，戚雅仙饰王秀英。此后经越剧广泛传播，成为多地剧种移植的喜剧经典。1960年香港长城电影公司将其拍摄为电影（夏梦、李嫱主演）。闽剧以此题材移植改编，剧情核心与越剧版本相近。
