# 夫人城

![cover](http://news.cnhubei.com/a/10001/202305/8b575423438c30ac37c6b8e2d373a41d.jpeg)


> 行当/题材：历史

## 剧情简介

该剧取材自三国时期东吴孙尚香（孙夫人）的历史与传说故事。孙权以和亲之计将刘备骗至江东，原拟借机图谋，却因孙尚香对刘备英雄气度产生倾慕，两人比武招亲，最终结为夫妻。刘备赴蜀后，在荆州修建"夫人城"将孙尚香软禁，不让其东归。孙尚香意欲携刘禅（阿斗）出走，被赵云奉诸葛亮之命在江边截下，孙尚香顾念阿斗安危，忍痛将其留下，孤身返回东吴。吴蜀开战之际，孙尚香夹于兄长孙权与丈夫刘备之间，进退两难，寸心如割。战后，孙尚香以吴使身份赴白帝城与刘备最后相见，促成吴蜀和好。全剧以孙尚香的悲剧命运为核心，呈现家国之间的两难与女性的坚韧。

（剧情核心通用，参照汉剧及史志资料综合呈现；闽剧版本演法见演出历史。）

## 主要角色

- **孙尚香**：东吴孙权之妹，刘备之妻，全剧主角，夹在国家与情感间的悲剧人物
- **刘备**：蜀汉君主，孙尚香之夫
- **孙权**：东吴之主，孙尚香之兄
- **赵云**：奉诸葛亮命截江，阻止孙尚香带阿斗东归
- **诸葛亮**：蜀汉军师，幕后决策者

## 故事来源 / 源流

取材自三国正史与民间传说中孙夫人（孙尚香）的故事，见《三国志》及民间故事"龙凤呈祥"等。以"夫人城"为题，强调孙尚香被囚禁这一历史细节，在汉剧等多个剧种中均有演出。汉剧新编版《夫人城》于2022年开排，以孙尚香为大女主叙事，获第三届全国戏曲（南方）会演闭幕演出。

## 备注

林颖是闽剧著名演员，24年专注闽剧传承创新，谢巧星系其师妹（同门刘小琴门下）。夫人城同名剧在汉剧、闽剧等多剧种中均有版本，三国题材在闽剧中较为少见，该剧具有一定的历史价值。
