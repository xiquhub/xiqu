# 安安送米

![cover](https://i.ytimg.com/vi/E44NcJ5nM-Q/maxresdefault.jpg)


> 行当/题材：家庭·孝道

## 剧情简介

东汉时期，书生姜诗侍母至孝，妻庞三春贤惠勤劳，一家和睦。邻妇秋香心地狡诈，暗中冒充三春之声诅咒婆母早死。姜母信以为真，将儿媳驱逐出门。三春流落在外，由姜诗的姑母收留，过着艰辛的日子。

年仅七岁的儿子安安，得知母亲被逐，心中悲痛。他每日上学时，偷偷将口粮省下几粒米，藏在路旁土地庙中，日积月累，将米粒凑成一碗，秘密送往母亲住处，以此表达对母亲的孺慕之情。这段"安安送米"的行为在乡间流传，感人至深，成为民间"一门三孝"的典范。

姜母后来患病，三春托人送去新鲜鱼肉表达孝心，邻妇秋香的恶行也随之被揭穿。姜诗方知妻子蒙冤，全家终于重归于好。全剧以孝道为核心，通过母子、夫妻之间的曲折情感，颂扬了传统家庭伦理中坚守善良、委曲求全的精神。

剧情核心通用，闽剧版本演法见演出历史。

## 主要角色

- **安安**：姜诗与庞三春之子，年仅七岁，故事核心人物，每日省米送母，展现童子纯孝
- **庞三春**（又作庞三娘）：姜诗之妻，贤淑孝顺，遭受冤屈仍不失善心
- **姜诗**：东汉书生，官至江阳县令，孝顺父母，轻信谗言误逐妻子
- **姜母**：姜诗之母，因信谗言而误解儿媳，后悔悟
- **秋香**：邻妇，反面人物，心地狡诈，制造家庭纷争

## 故事来源 / 源流

本剧取材于中国古代民间孝道传说"安安送米"，主人公安安是历史上东汉孝子，其孝行后被元人郭居敬辑入《二十四孝》。明代传奇《跃鲤记》亦演绎此题材。各地方戏曲（潮剧、闽剧等）均有改编版本。故事以"一门三孝"为主题，歌颂家庭伦理与孝道精神。
