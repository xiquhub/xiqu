# 铜雀台

![cover](https://image.baidu.com/search/down?url=https://tvax1.sinaimg.cn/large/ea98bce0gy1h12fowgwixj21hc0xc116.jpg)


> 行当/题材：历史人情剧 / 生、旦、净行

## 剧情简介

铜雀台为曹操所筑，坐落于邺城，是三国时代权力与才情交汇之所。本剧以此为背景，深入曹家内部，呈现曹操、曹丕、曹植与甄宓四人之间错综复杂的情感纠葛。

曹操雄才大略，却也是一个感情复杂的父亲；长子曹丕性格沉稳工于心计，有帝王之志；次子曹植才华横溢，洒脱不羁，却命运坎坷；甄宓为袁绍之子袁熙之妻，兵败后落入曹家，与曹植相互仰慕，却被曹丕强娶为妻。四人之间，政治与情感交织，父子兄弟之伦与男女爱恨缠绕，在理想、权力与情感的激烈碰撞中，每个人都付出了沉重代价。全剧颇具悲剧色彩，在历史史诗的框架内探讨人性的复杂。

## 主要角色

- 曹操：乱世枭雄，铜雀台的缔造者
- 曹丕：长子，心机深沉，与曹植争嫡
- 曹植（字子建）：才子，作《洛神赋》以寄情
- 甄宓：命运多舛的女性，兼受曹植爱慕与曹丕占有

## 故事来源 / 源流

以三国历史（汉末建安年间）人物为素材改编，结合《三国志》及裴注、《洛神赋》等文学传说。曹植甄宓之恋属历史文学传说，多部文艺作品以此为题材演绎。本剧为闽剧新编历史剧，编剧安志强，导演天博、陈建军。
