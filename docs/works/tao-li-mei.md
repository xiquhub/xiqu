# 桃李梅

![cover](http://image2.sina.com.cn/ent/2004-12-19/1103428160_N2aabV.jpg)


> 行当/题材：社会伦理喜剧 / 旦、生行

## 剧情简介

本剧以袁家三姐妹为核心，展开一段正义战胜权贵的喜剧故事。

集宁知县袁如海软弱无能，其女分别为玉桃（长女，已出嫁）、玉李（次女，已与颜文敏订婚）和玉梅（三女，聪慧果敢）。当地总兵方亨行倚仗权势欺压百姓，竟逼迫知县将已有婚约的玉李纳为小妾。母亲封氏与三女玉梅合谋应对，设计取得方亨行罪证。玉梅随即女扮男装进京告状，途中偶遇进京赶考的举子洪学勤，两人相互赏识，同赴科场，双双金榜题名——洪学勤中状元，玉梅高中后被封八府巡按。归来之日，玉梅升堂审案，欲擒故纵，将方亨行治罪惩办。大团圆收场：玉李与颜文敏喜结连理，玉梅与洪学勤成婚，袁府双喜临门。

## 主要角色

- 玉梅：三女，主角，女扮男装、考中巡按的奇女子
- 玉李：次女，被强逼婚嫁的受害者
- 洪学勤：举子，与玉梅同榜，玉梅的良配
- 封氏：袁如海之妻，助女抗争
- 方亨行：总兵，倚仗权势的反面人物
- 袁如海：知县，软弱父亲

## 故事来源 / 源流

本剧改编自吉剧《桃李梅》，原剧由王肯编剧，1960年首演于吉林省，此后三十年演出逾三千八百场。该剧以女扮男装、告状申冤、科举功名为结构支撑，被数十个剧种移植，越剧、评剧、豫剧、秦腔等均有版本。福建省芳华越剧团亦曾以越剧形式在福州演出此剧（2016年）。闽剧版本系在上述广泛流传的基础上移植改编，剧情核心通用。
