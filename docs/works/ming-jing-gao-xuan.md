# 明镜高悬

> 行当/题材：净（包公）／公案清官

## 剧情简介

《明镜高悬》为包公断案题材剧目，以"明镜高悬，铁面无私"为主旨，讲述包拯重审旧案、直面自身失误，最终秉公判决的故事。

剧情核心通用：包拯在任知府时曾办一宗命案，二十余年后有人为蒙冤者鸣冤，重新诉至开封府。包公不护旧名声，毅然重启旧案，仔细查访，发现当年确有冤情，经历内心挣扎后，仍不顾一生清誉，坚持公正审判，最终平反冤案，惩处真凶。剧名"明镜高悬"取古代官府匾额之意，象征清廉公正的司法精神。

闽剧版本以福州方言演绎，唱腔保留包公戏惯常的激越豪放风格，重点渲染审案时的紧张对峙与包拯内心的道德抉择，属包公戏中兼具自省色彩的少见类型。

## 故事来源 / 源流

包公故事源自北宋名臣包拯（999—1062）的历史形象，经宋元话本、明清小说《三侠五义》及续书不断丰富，形成庞大的包公戏系列，流传于京剧、豫剧、闽剧等数十个剧种。"明镜高悬"典故出自古代公堂匾额，明代已广泛用于公案文学，此剧当属该系列的独立折子戏。
