# 梨花情

![cover](http://www.yuejuopera.net/uploads/allimg/160818/1-160QQ10U2V8.jpg)


> 行当/题材：才子佳人

## 剧情简介

（剧情核心通用，本剧存在粤剧、越剧、评剧、晋剧、豫剧等多个版本，闽剧版细节未见专项记录。）

花乡绣女梨花与豪门千金冷艳、春江秀才孟云天、钱塘富商钱友良，四人在各自追求婚姻幸福的过程中，共同面对"情"与"钱"的两难抉择。

孟云天与梨花情投意合，两人不顾父母之命，携手私奔，在清贫的日子中相濡以沫。然而孟云天在贫困面前逐渐变得懦弱退缩，最终以"门不当户不对"为由，抛弃梨花，另娶豪门千金冷艳。

梨花伤心欲绝，投河自尽，幸被诚实忠厚的商人钱友良救起。二人日久生情，结为夫妻，共同打拼。不料一场大火将他们的家业付之一炬，生活再度跌入困境。

此时，冷艳已死，重新富贵的孟云天再度想起梨花的好，试图旧情复燃。梨花历经磨难，已深知什么才是真挚的情感，在情与钱的面前，毅然选择了与钱友良共守患难。

## 主要角色

- **梨花**：花乡绣女，善良贤慧，历经情变与磨难，最终选择真情
- **孟云天**：春江秀才，软弱自私，弃贫娶富，落个孤苦收场
- **冷艳**：豪门千金，嫁给孟云天，早逝
- **钱友良**：钱塘商贾，诚实厚道，救梨花、与之共苦共甘

## 故事来源 / 源流

1997年由编剧包朝赞创作的新编剧本，以"情与钱"为主题展开叙事，被多个地方戏曲剧种移植。
