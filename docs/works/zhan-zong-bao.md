# 斩宗保

![cover](http://www.fj.chinanews.com/2022/0213/U969P4DT20220213134053.jpg)


> 行当/题材：历史/英雄/武戏

## 剧情简介

《斩宗保》取材于北宋杨家将系列故事，围绕杨宗保之死展开。杨宗保是杨延昭（杨六郎）之子、穆桂英之夫，是杨门将门中的重要人物。

北宋边疆告急，杨宗保奉命领兵出征，在征战中遭遇敌军伏击，部下有人通敌，致使杨宗保陷入重围，最终力战身亡。消息传回，穆桂英悲痛万分，佘太君率杨门众将痛定思痛，继续抵御外侮。"斩宗保"之名或指某一版本中误斩宗保的戏剧性情节，也可能指宗保被敌斩杀的悲壮场面，具体细节因版本不同而有出入。

该剧着力塑造杨宗保忠勇报国的英雄形象，同时烘托杨门女将的悲壮气概，是杨家将题材中着力表现"壮烈牺牲"主题的折子戏之一。剧情核心通用（取自杨家将共同故事传统）。

## 主要角色

- **杨宗保**：杨延昭之子，穆桂英之夫，武将
- **穆桂英**：杨宗保之妻，女将
- **佘太君**：杨家老祖宗，统领全家
- **杨延昭**：杨六郎，杨宗保之父

## 故事来源 / 源流

改编自明代小说《杨家府世代忠勇通俗演义》及相关民间故事，历代多个剧种（京剧、豫剧、评剧、越剧等）均有杨宗保相关剧目，闽剧《斩宗保》为本地化版本。
