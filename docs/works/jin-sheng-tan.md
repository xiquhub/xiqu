# 金圣叹

![cover](https://pic.app.fqlook.com/forum/20230223173653front1_0_155891_Fu2xdKfd3lJ5CxGG2FZDaQXfMgPL.jpg)


> 行当/题材：历史

## 剧情简介

本剧以明末清初著名文学评点家金圣叹的真实生平为素材，聚焦其晚年遭遇的政治风波与慷慨赴死的壮烈结局。

金圣叹（1608—1661），原名金人瑞，苏州吴县人，以评点《水浒传》《西厢记》等"六才子书"名噪一时，是清代最具个性的文人之一。他恃才傲物，诙谐机敏，以犀利笔锋横扫文坛。

顺治十八年（1661年），皇帝驾崩，新皇即位。苏州吴县县令贪污暴虐，横征暴敛。金圣叹参与士人聚集哭庙一事，被当局扣上"聚众哗变"的罪名，以叛逆之罪论处，被判斩首。

临刑前，金圣叹从容不迫，向刽子手揭示藏于耳中的纸条——上书"好"字——笑称人头落地、耳分两边，乃人生一大乐事。据记载，他留给两个儿子的最后一副对联为"莲子心中苦，梨儿腹内酸"，以谐音寄托对子女的无限深情。

全剧通过金圣叹的才情与不屈，折射专制政治对文人精神的戕害。

## 主要角色

- **金圣叹**：主人公，明末清初文学评点家，因哭庙案被斩，临死仍显幽默才情
- **吴县知县**：贪官，横征暴敛，激起民愤
- **金圣叹之子**：临别接受父亲绝命联，沉默哀泣

## 故事来源 / 源流

依据历史人物金圣叹生平改编，史有实据。"哭庙案"见于清代史料，金圣叹被执于1661年8月7日。其临刑绝语与留子诗联均见于野史笔记，流传甚广。
