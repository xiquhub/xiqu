# 卷席筒

## 剧情简介

（剧情核心通用，见于河南曲剧等多个版本，闽剧版细节有所出入。）

曹林继弦赵氏为独吞家产，将前妻之子曹宝山一家赶出正院，逐入下院受尽白眼。少年苍娃（小苍娃）是曹宝山的小叔，心地善良，暗中接济兄嫂，卷着席筒悄悄送去粮食。曹宝山发愤读书，苍娃千方百计资助其进京赶考。

赵氏心狠手辣，毒杀亲夫曹林后，将罪名嫁祸于曹宝山之妻张氏。张氏含冤入狱，被判死刑。苍娃不忍嫂嫂受冤，挺身而出替罪，甘愿以一己之命换嫂嫂生还。

就在苍娃即将被处决之际，曹宝山金榜题名高中状元，返乡后昭雪冤案。苍娃终于获救，一家人破镜重圆。

全剧以苍娃这一机灵善良、侠义担当的少年形象为核心，讽刺恶毒继母，歌颂手足之情与牺牲精神。

## 主要角色

- **曹张苍（小苍娃）**：主人公，少年，机智灵动，侠义热肠，替嫂赴死
- **张氏**：曹宝山之妻，贤惠善良，含冤入狱
- **曹宝山**：曹林前妻之子，读书进京，中状元后平反冤案
- **赵氏**：曹林续弦，反面人物，毒夫嫁祸

## 故事来源 / 源流

本剧为多剧种共有传统剧目，以河南曲剧版本最为著名（海连池主演小苍娃）。各地剧种包括豫剧、吕剧、闽剧等均有移植演出。故事情节与"继母害子"类民间叙事模式相近，具体原始来源未见明确记载。
