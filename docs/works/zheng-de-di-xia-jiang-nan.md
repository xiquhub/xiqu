# 正德帝下江南

![cover](https://i.ytimg.com/vi/5-KtzDxDQPA/hqdefault.jpg)

## 剧情简介

《正德帝下江南》取材于明武宗朱厚照南巡的历史故事，属"皇帝微服私访"类传统戏曲题材。明武宗正德皇帝以个性张扬、喜好微行著称，历史上其南巡之举曾引发朝廷内外激烈争议。

戏曲化的《正德帝下江南》通常描写朱厚照以平民身份乔装下访，沿途经历种种奇遇与风流韵事——或在市井中广结豪侠，或与民女邂逅情定，或惩治欺压百姓的官吏。剧中皇帝一反庙堂威严，以普通人的面目游历江南烟雨，既展现了民间风情的生动，也通过皇帝化身草民的视角达到讽谕效果。全剧以喜剧笔调呈现，多有插科打诨，是传统"帝王出巡"类喜剧的闽剧版本。

歌仔戏等剧种亦有《正德皇帝游江南》剧目留存，足见此类故事在福建及台湾闽南语戏曲圈中流传之广。

## 主要角色

- **朱厚照（正德帝）**：明武宗，微服出游的皇帝，放浪形骸而有情有义
- **随行臣属**：陪伴皇帝出巡的侍从或近臣
- **江南民女/官宦之女**：与皇帝邂逅的女性角色

（具体角色名称依版本不同而异。）

## 故事来源 / 源流

源自明代历史上明武宗朱厚照南巡的史实与民间传说，经历代说书、弹词、戏曲等多种艺术形式演绎，逐渐形成固定的"正德游江南"故事模式。各地戏曲、歌仔戏均有改编版本。
