# 裴秀英四告

![cover](https://jcdn.xhby.net/JHD/publish/20230618/1687083915650046064.jpg)


> 行当/题材：旦（苦旦）／公案苦情

## 剧情简介

剧情核心通用（据淮海戏《皮秀英四告》系列版本综合改写，闽剧版题为"裴秀英"，人名或有出入）：

退任阁老黄彦忠嫌贫爱富，强行逼迫女婿李贤贵退婚。此事被更夫皮盾偷听在耳，皮盾趁机假冒李贤贵，杀死丫环，劫走财物，嫁祸于人。黄彦忠买通知县戈自明，对李贤贵严刑逼供，使其屈打成招，蒙冤入狱。

李贤贵之嫂皮秀英（闽剧本或作裴秀英）得知真相，赶赴县衙为其伸冤，却被衙役乱棍打出。走投无路之下，她恳请亲兄皮盾陪同进京申诉，不料皮盾正是真正的杀人凶手，途中借马匹为名，用缰绳将皮秀英勒死于白马上，独自回乡谎称她已身亡。

皮秀英大难不死，在荒野苏醒，只身辗转入京，于路上拦轿鸣冤：一告阁老黄彦忠挟势枉法，二告知县戈自明受贿枉判，三告兄长皮盾杀人嫁祸，四告丈夫（或夫家）李贤贵未能自明清白。蹊跷的是，轿内之人竟是已经飞黄腾达、手握重权的李贤贵本人。大堂之上妻子状告丈夫，悲喜交加，最终水落石出，冤案昭雪，诸恶人一一受到惩处。

## 主要角色

- **裴秀英（皮秀英）**：苦旦，刚烈坚韧，独身告状的女主角
- **李贤贵**：裴秀英夫家，蒙冤者，后飞黄腾达
- **皮盾**：裴秀英之兄，反派，真正的杀人凶手
- **黄彦忠**：退任阁老，嫌贫爱富，主要反派之一
- **戈自明**：贪腐知县，受贿枉判

## 故事来源 / 源流

改编自淮海戏传统剧目《皮秀英四告》，该剧在苏北地区几乎家喻户晓，并被扬剧、楚剧、花鼓戏等多个剧种移植演出。闽剧版本将主角名字改为"裴秀英"，当属地方化改编。
