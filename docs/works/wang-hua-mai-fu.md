# 王华买父

![cover](https://i.ytimg.com/vi/RP74TeqyhO0/hqdefault.jpg)


> 行当/题材：孝义传奇剧 / 生、旦、老生行

## 剧情简介

本剧源自民间流传广泛的孝义故事，剧情核心通用。

宋代渔夫王华，家境清贫，心地善良。一日在街头集市，见一位衣衫褴褛的老翁张贴告示，自称年老无儿无女、无人奉养，愿"卖身作父"，以求有人赡养。王华被老翁的可怜处境所动，不顾家中拮据，慷慨买下老翁，接回家中，奉为亲父，倾尽家产侍奉，甚至变卖妻子的金钗首饰，为老人购买山珍海味以尽孝道。

妻子起初难以理解，却被丈夫的真诚孝心感动，合力赡养。日子一久，老翁真实身份逐渐浮出水面——他原是北宋皇室"八千岁"赵德芳（八王爷），微服私访以察民情。感念王华夫妇的大义仁心，老翁（八王爷）赐予王华一家荣华富贵，善有善报，大团圆收场。

## 主要角色

- 王华：主角，善良渔夫，买老翁为父
- 王华妻：善解人意，与丈夫共同尽孝
- 老翁（赵德芳/八千岁）：皇族亲王，微服私访，真实身份后来揭晓

## 故事来源 / 源流

源自宋代民间传说，与北宋皇室赵德芳（"八贤王"）相关故事体系交织。故事以孝义善报为主题，在民间广泛流传，被改编为评剧（又名《回龙传》《由命不由人》）、越剧、豫剧、河北梆子、吕剧、淮剧等多个剧种版本，亦有评书、弹词等曲艺形式演绎。
