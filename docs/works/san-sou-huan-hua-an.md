# 三搜幻化庵

![cover](https://i.ytimg.com/vi/lUG7lPmeCc8/maxresdefault.jpg)


> 行当/题材：传奇 / 寻亲戏 / 爱情悲剧

## 剧情简介

《三搜幻化庵》又名《玉蜻蜓》，是一部讲述跨越十六年寻亲情缘的传奇剧目。故事以幻化庵为中心场景，描绘了一段因爱生恨、因血缘割舍不断的家族悲欢。

尼姑王志贞本是出家之人，却与风流秀才申贵升相遇，二人情投意合，申贵升进庵与志贞秘密结为夫妻，此事引来外界三度搜庵查办。申贵升不幸早逝，志贞已怀身孕，产下一男婴后，迫于庵规处境，忍痛将孩子送出，仅留下一枚玉蜻蜓和一封血书作为日后相认的信物。

十六年后，孩子已长大成人，高中举人。一次偶然，他发现了母亲留下的血书，方知身世之谜，遂立志寻访生母。历经辗转，母子终得相认，将生父申贵升的灵牌迎回家中。养母与生母之间的恩怨情仇，也在真相大白后逐步化解，最终以"合家团聚"收尾。

## 主要角色

- **申贵升**：秀才，生父，早逝
- **申贡生**（举人子）：长大后寻亲的主人公

## 故事来源 / 源流

本剧源自传统剧目《玉蜻蜓》，该故事在苏州弹词、昆剧、沪剧、越剧等多个剧种中均有流传，以"玉蜻蜓"这一信物为纽带串联寻亲情节。闽剧版题名《三搜幻化庵》，强调剧中三次搜庵的关键冲突。
