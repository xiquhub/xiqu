# 探花披龙袍

> 行当/题材：宫廷历史剧 / 生、净、老旦行

## 剧情简介

本剧与"狸猫换太子"系列故事及包拯题材密切相关，剧情核心通用。

宋代，书生赴京赶考，因机缘巧合与皇室纠葛发生交集——一说探花郎误穿龙袍遭受牵连，或与包拯断案惩奸相关联。据"打龙袍"系列故事脉络：宋仁宗之生母李后曾遭陷害流落民间，陈琳道出当年"狸猫换太子"真相后，仁宗恍然大悟，命包拯代打龙袍以示认罪谢母。本剧以"探花"切入，或演绎科举功名与宫廷恩怨交织的故事，探花郎因缘际会披上龙袍，卷入皇家秘闻。
