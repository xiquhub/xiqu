# 六月雪

> 行当/题材：公案

## 剧情简介

本剧改编自元杂剧《窦娥冤》，闽剧版通行剧情如下：孝女窦娥自幼丧母，父亲窦天章因债务将她卖给蔡婆做童养媳。窦娥与蔡婆之子蔡昌宗成婚不久，昌宗便上京赴考，留下孤儿寡母相依为命。

地方恶霸张驴儿觊觎窦娥，以毒药谋害蔡婆，不料毒死了自己的母亲，反借此反诬窦娥婆媳毒害人命。知府被收买，不辨是非将窦娥屈打成招，判处死刑。行刑当日，窦娥含冤立誓：血溅白练、六月飞雪、楚州大旱三年。三誓随即应验，六月飘雪震惊天地。

蔡昌宗金榜题名，任陕西道台，得知此事后亲回重审此案，真相大白，窦娥沉冤昭雪，张驴儿等奸人受到惩处。

## 主要角色

- **窦娥**：女主角，含冤受死的孝女
- **窦天章**：窦娥之父，因贫穷将女儿卖给蔡家
- **蔡婆**：婆婆，与窦娥相依为命
- **蔡昌宗**：窦娥夫婿，高中状元后为妻昭雪
- **张驴儿**：地方恶霸，奸人反诬主谋

## 故事来源 / 源流

源自元代关汉卿杂剧《感天动地窦娥冤》，取材于汉代"东海孝妇"民间传说。《六月雪》是《窦娥冤》在各地方剧种中流传的通称，闽剧版在情节上与元杂剧原著略有出入，结局普遍改为夫妻重逢、冤情昭雪。
