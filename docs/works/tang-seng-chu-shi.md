# 唐僧出世

![cover](https://www.fzmjtc.cn/__local/4/B1/F5/AA7A86C08D94ED8A6EE542FF735_407E029B_278FC.jpg)


> 行当/题材：神话传奇剧 / 生、旦兼用

## 剧情简介

本剧演绎《西游记》中唐僧出世的前传故事，剧情核心通用。

唐贞观年间，海州学子陈光蕊高中状元，与宰相之女殷温娇成婚，随即奉旨赴任江州知州。夫妻水路赴任途中，渡船水寇刘洪见财起意，又垂涎殷温娇姿色，将陈光蕊杀害推入江中，假冒其身份赴任为官，强逼温娇同居。殷温娇知丈夫已死，却为腹中胎儿忍辱含垢。临产后将婴儿包裹血书，放于木板漂流水上，婴孩为金山寺法明长老所救，取名"江流儿"，抚育成人，后赐法名陈玄奘。

十八年后，玄奘查清父仇，借龙王之力寻得陈光蕊（龙宫保护），最终家仇得报。然殷温娇因失身侍贼，有愧于丈夫，家人团圆之际投江自尽，徒留唐僧大悲。此后唐僧承受佛法，踏上取经之路。

## 主要角色

- 陈光蕊：唐僧之父，状元郎，遭难沉江
- 殷温娇：唐僧之母，宰相女儿，忍辱负重
- 刘洪：凶徒，杀害陈光蕊、冒官夺妻
- 法明长老：金山寺住持，收养婴儿江流儿
- 江流儿（陈玄奘）：即唐三藏，本剧主角

## 故事来源 / 源流

《西游记》第九回"唐僧出世"情节，据考系清初汪澹漪《西游证道书》增插，明嘉靖本《西游记》原无此回。故事更早源头为宋元南戏《陈光蕊江流和尚》及元杂剧《唐三藏西天取经》，是《西游记》成书前即广泛流传的独立传说。各地戏曲以此为独立折子戏或连台本演出。
