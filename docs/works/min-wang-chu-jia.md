# 闽王出家

> 行当/题材：历史传奇

## 剧情简介

本剧取材于"开闽三王"之首王审知（862—925）的历史传说。王审知为五代时期闽国的开国君主，史称"开闽王"，以轻徭薄赋、笃信佛法著称，深受福建民间崇奉，被尊为"开闽圣王"。

"闽王出家"或为民间传说中王审知晚年舍弃王位、皈依佛门的戏剧化演绎，也可能涉及其子孙礼佛出家的故事。具体剧情细节因资料有限，尚待进一步考证。

以王审知史实及福州民间信仰为素材。王审知笃信佛教，晚唐五代时期在福州广建寺庙，民间围绕其虔诚信仰衍生出诸多传说，"出家"极可能是其中之一的戏剧化改编。
